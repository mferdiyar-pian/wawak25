## 🚀 Tentang Project

**HBD WAWAK 25** adalah website sederhana namun bermakna yang dibuat untuk memberikan ucapan ulang tahun secara digital dengan tampilan yang menarik dan interaktif.

Project ini dibangun menggunakan **Laravel Framework**, sehingga memiliki struktur yang rapi, scalable, dan mudah dikembangkan.

---

## ✨ Fitur Utama

- 🎂 Tampilan ucapan ulang tahun yang menarik  
- 🎨 UI sederhana & elegan  
- ⚡ Performa cepat dengan Laravel  
- 🌐 Bisa diakses online (live project)  

---

## 🛠️ Teknologi yang Digunakan

- **Laravel** — Backend Framework  
- **Blade Template** — Templating Engine  
- **HTML, CSS, JavaScript** — Frontend  
- **Hosting Online** — Deployment
